# Getting started

Before proceeding, you'll need to have the last stable NodeJS (min version 20), npm, and yarn installed on your machine.

It's highly recommended to use [nvm-windows](https://github.com/coreybutler/nvm-windows#node-version-manager-nvm-for-windows) to switch Node versions between different projects. Also make sure to have [BitBucket](https://bitbucket.org/) active and linked to your Marex account.
