# Libraries and dependencies

The component library is built leveraging on the latest trends with periodic updates. The optional dependencies are used to enrich Neon apps with new features.

## Core dependencies

- `react` - Core
- `vite` - Frontend tooling: bundler, dev server, etc.
- `@emotion/react` - Styling engine
- `@mui/material` - Base theme components
- `@mui/system` - Base theme system utilities
- `@mui/icons-material` - MUI icons
- `@mui/lab` - Base theme components

## Utilities

- `date-fns` - Utils to parse date and time
- `numeral` - Utils to format numbers
- `react-hook-form` - Form validation
- `zod` - Schema validation library for `react-hook-form`

## Other components

- `react-select` - Select component
- `simplebar-react` - Replace native scrollbars with custom ones
- `nprogress` - Display page loading progress
- Froala - Rich text editor (WYSIWYG)
- AG-Grid - Data grid for complex scenarios
- React Dropzone - File upload with drag and drop support
- Helmet - SEO support
- Apex Charts - Charting library
